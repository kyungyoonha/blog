## 1. 사용하는 이유?

-   리덕스를 사용하는 어플리케이션의 경우, 타입 / 액션 / 리듀서 를 분리하여 폴더로 관리하는데
-   하나의 기능을 수정할 때는 이 기능과 관련된 여러 개의 파일을 수정해야 하는 일이 생긴다.
-   이러한 불편함을 개선하기 위해 나옴
-   Reducer 파일 안에 액션타입과 액션 생성자 함수를 함께 넣어서 관리하고 이를 '모듈' 이라고 한다.
-   기능 별로 수정하기가 편하다.
